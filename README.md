# Health Force Academy

## Healthcare Career Counselling Platform for Students After Class 12

A modern, production-ready web application built with Next.js, React, TypeScript, and Tailwind CSS.

### Features

- 🎯 Premium Landing Page with Hero Section
- 📱 Multi-step Lead Generation Form
- 🤖 AI Chatbot Integration
- 📊 Admin Dashboard for Lead Management
- 👥 Partner Management System
- 🎓 Comprehensive Course Catalog
- 📈 Lead Routing & Assignment
- 🔐 Supabase Integration
- ⚡ Optimized Performance (90+ Lighthouse)
- 🎨 Glassmorphism Design
- ✨ Smooth Animations with Framer Motion
- 📱 Fully Responsive Design
- 🔍 SEO Optimized

### Tech Stack

- **Framework**: Next.js 14
- **UI Library**: React 18
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Components**: shadcn/ui
- **Animations**: Framer Motion
- **Database**: Supabase

### Getting Started

```bash
npm install
cp .env.example .env.local
npm run dev
```
